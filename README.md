## ğŸ“¦ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/NoobSambit/terminal.sambitpradhan.git
   cd terminal.sambitpradhan
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. **Build for production**

   npm run build
